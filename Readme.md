# Personal Academic Website

Website portofolio akademik yang menampilkan profil profesional, riwayat pendidikan, pengalaman penelitian, publikasi, proyek, serta bidang keahlian di bidang Artificial Intelligence, Machine Learning, Deep Learning, Computer Vision, Data Science, dan Natural Language Processing.

Website ini dikembangkan sebagai media untuk memperkenalkan aktivitas akademik, penelitian, pengajaran, serta berbagai proyek pengembangan perangkat lunak. Selain sebagai portofolio profesional, website ini juga menjadi sarana untuk membangun kolaborasi penelitian, berbagi publikasi ilmiah, dan mendokumentasikan berbagai kontribusi di bidang ilmu komputer.

## Features

- Responsive Academic Portfolio
- Professional Research Profile
- Curriculum Vitae (CV)
- Research Portfolio
- Publications
- Teaching Portfolio
- AI & Software Projects
- Expertise & Services
- Collaboration & Contact Information
- Mobile Friendly Navigation
- Modern UI/UX Design

## Technologies

- Bootstrap Icons
